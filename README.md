# CodexLooper

CodexLooper is a local autonomous development loop that connects:

- MEX for selective project memory;
- Ralphex for roadmap execution, retries, reviews and commits;
- CloseRouter for controlled model access;
- Terra as the default builder;
- Sol as the independent reviewer;
- deterministic tests and Git as objective gates.

Status: WP0 integration verification.

Version 1 is a local CLI. It does not send messages, deploy to production, merge pull requests or perform other external actions without explicit authorization.
